# 📚 Bookly Pro - Personal Book Inventory

ระบบจัดการคลังหนังสือส่วนตัวที่มาพร้อมกับดีไซน์ระดับพรีเมียมและอนิเมชันที่ลื่นไหล

### ✨ Features
- **Modern UI:** ดีไซน์สไตล์ Glassmorphism อ่านง่าย สบายตา
- **Animations:** ใช้ Staggered Load และ Subtle Blobs เพื่อประสบการณ์ใช้งานที่ดีที่สุด
- **Authentication:** ระบบสมัครสมาชิกและล็อคอินที่ปลอดภัย
- **Full-stack:** ขับเคลื่อนด้วย Next.js 15, Prisma และ Tailwind CSS

### 🚀 Tech Stack
- [Next.js](https://nextjs.org/)
- [Prisma](https://www.prisma.io/) (SQLite)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide Icons](https://lucide.dev/)
